三、 遍历（迭代）方法

1.forEach()

对数组中的每一项运行指定的函数。这个方法返回undefined，即使你return了一个值。

Array.forEach()参数语法：

element index array
当前元素 当前元素的索引 (可选) 数组本身(可选)

Array.forEach()不能中断循环(使用break，或continue语句)。只能用return退出本次回调，进行下一次回调。
